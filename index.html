PROYECTOS

Sistema de inventario

Detalles del proyecto

Este sistema fue diseñado específicamente para la gestión eficiente de una bodega. Cuenta con módulos de ventas, compras e inventario que permiten registrar tanto la entrada como la salida de productos disponibles en stock. La información se almacena de manera segura en una base de datos PostgreSQL, garantizando un control preciso y actualizado de los movimientos de mercancías.

TECNOLOGIAS: Visual Basic y PostgreSQL

MultiWhatsapp

Detalles del proyecto

La aplicación MultiWhatsApp fue desarrollada para abordar la necesidad de gestionar múltiples cuentas de WhatsApp de forma simultánea en entornos empresariales. Esta solución permite a empleados y empleadores utilizar diferentes instancias de WhatsApp, asignadas a distintos tipos de asistencia o áreas de trabajo, mejorando significativamente la comunicación interna y facilitando la colaboración entre equipos
